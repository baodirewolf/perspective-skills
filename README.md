## ⚠️ 重要声明

### 学术研究定位
本技能集基于 **[Nuwa Skill 造人术](https://github.com/openclaw/openclaw/tree/main/skills/nuwa-skill)** 方法论，旨在**提取历史人物的思维框架**，用于：
- ✅ 学术研究与文化传承
- ✅ 思维方法学习
- ✅ 跨学科思考训练

### 免责声明

**倪海厦技能**：
- 所有中医内容**不代表医疗建议**，如有健康问题请咨询正规医疗机构
- 命理玄学内容**仅供了解传统思维方式**，不涉及实际预测服务
- 使用者应理性对待，**避免迷信，相信科学**

**南怀瑾技能**：
- 涉及宗教内容**仅为哲学思想研究**，不代表任何宗教立场
- **不涉及风水命理等迷信活动**，不涉及实际预测服务
- 使用者应以**学术研究态度**学习，避免盲目崇拜

**荣格 & 阿德勒技能**：
- 心理学内容**不能替代专业心理咨询**
- 梦境分析等仅供自我探索参考

## 🎯 使用指南

### 倪海厦 · 中医与玄学

**触发词**：`倪海夏`、`玄学`、`八字命理`、`铁板神数`、`中医`、`紫微斗数`、`紫微斗术`

**使用场景**：
- 中医健康咨询（经方、诊断、治疗）
- 命理预测（紫微斗数、八字、风水）
- 五术合一综合判断

**示例**：
```
用户：我睡眠不好，倪海夏会怎么看？
→ 激活倪海厦视角，从六经辨证、阳气角度分析

用户：帮我看看八字
→ 从命盘三层次（先天/后天/流年）分析
```

**核心心智模型**：
1. 阴阳为纲，阳气为本
2. 经方为根本，六经辨证
3. 命理三层次模型（先天/后天/流年）
4. 三分之一法则
5. 五术合一

---

### 南怀瑾 · 国学智慧

**触发词**：`南怀瑾`、`国学`、`儒释道`、`三教合一`、`经史合参`、`禅修`、`佛学`

**使用场景**：
- 人生困惑与修身养性
- 儒释道经典解读
- 文化传承与心灵成长

**示例**：
```
用户：工作焦虑怎么办？
→ 从三教圆融角度：儒（担当）、道（放下）、佛（空性）

用户：怎么理解《论语》这段话？
→ 经史合参，结合历史背景解读
```

**核心心智模型**：
1. 儒释道三教合一
2. 经史合参方法论
3. 心源学说
4. 内修外用实践论

---

### 荣格 · 深度心理学

**触发词**：`荣格`、`Jung`、`心理学`、`梦境`、`无意识`、`原型`、`集体无意识`、`个体化`

**使用场景**：
- 梦境分析与解读
- 自我探索与心理成长
- 原型与象征理解

**示例**：
```
用户：我做了个奇怪的梦
→ 从象征化思维分析梦境的无意识内容

用户：怎么理解我的阴影？
→ 运用原型系统模型解释阴影整合
```

**核心心智模型**：
1. 心灵结构三层模型
2. 原型系统模型
3. 个体化过程模型
4. 象征化思维模型

---

### 阿德勒 · 个体心理学

**触发词**：`阿德勒`、`Adler`、`个体心理学`、`自卑与超越`、`目的论`、`课题分离`、`社会兴趣`

**使用场景**：
- 人际关系困扰
- 自卑感处理与超越
- 家庭教育与亲子问题

**示例**：
```
用户：孩子不想上学怎么办？
→ 用目的论分析：孩子想通过「不上学」达到什么目的？

用户：我总是很自卑
→ 从自卑与超越模型，将自卑转化为成长动力
```

**核心心智模型**：
1. 目的论模型（由未来目标驱动）
2. 自卑与超越模型
3. 社会兴趣模型
4. 课题分离

| 检查项 | 验证问题 | 通过标准 |

## 🧠 蒸馏方法论

这些Skill基于 **Nuwa Skill 造人术** 方法论创建：

### Phase 1: 六维研究
- 著作论文（系统思维）
- 长对话/访谈（即兴思维）
- 社交媒体（表达DNA）
- 外部评价（批评与盲区）
- 决策记录（言行一致性）
- 时间线（思想演变）

### Phase 2: 框架提取
- 核心心智模型（3-10个）
- 决策启发式（5-10条）
- 表达DNA分析
- 诚实边界定义

### Phase 3: Skill生成
- 按标准模板生成 SKILL.md
- 质量验证（已知测试、边缘测试、声音测试）
